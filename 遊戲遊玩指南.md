# 🎮 Urgency - 遊戲遊玩指南

## 📖 遊戲概述

**Urgency** 是一款 **2D 俯視角生存恐怖遊戲**。玩家需要在充滿殭屍的黑暗環境中求生，管理有限的資源，解開謎題，最終逃離危險區域。

---

## 🎯 遊戲目標

1. **存活** - 避免被殭屍殺死
2. **探索** - 尋找鑰匙、彈藥和補給品
3. **逃脱** - 到達關卡終點（Exit）完成關卡

---

## 🕹️ 操作方式

|------|------|

| 射擊/攻擊 | 滑鼠左鍵 |
| 互動（開門/撿物品） | E 或 F |

| 操作 | 說明 |
|------|------|

## ❤️ 玩家狀態

### 生命值 (Heart)
- 預設最大值：100
- 被殭屍攻擊會扣血
- 血量歸零 = 遊戲結束（進入死亡畫面）
- 會緩慢自動回復

### 護甲 (Armor)
- 吸收部分傷害
- 護甲優先消耗，再扣生命值
- 不會自動回復，需找護甲補給

### 電池 (Battery Power)
- 手電筒的能源
- 持續消耗
- 電池耗盡 = 手電筒熄滅
- 需撿取電池補充

### 煤油 (Kerosene)
- 提燈的燃料
- 持續消耗
- 需撿取煤油補充

## 🔫 武器系統

### 遠程武器
| 武器類型 | 彈藥類型 | 特點 |
|----------|----------|------|
| 手槍 (Pistol) | 手槍子彈 | 基礎武器，適合單體 |
| 步槍 (Rifle) | 步槍子彈 | 高傷害，射速快 |
| 弓箭 (Bow) | 箭矢 | 安靜，不驚動敵人 |

### 近戰武器
- 可進行近距離攻擊
- 不消耗彈藥
- 傷害較低，風險較高

### 彈藥管理
- 彈藥有限，需謹慎使用
- 可從場景中撿取彈藥包
- 擊殺敵人有機率掉落彈藥

## 👹 敵人類型

### 普通殭屍 (Zombie)
| 屬性 | 說明 |
|------|------|
| 攻擊方式 | 近戰 |
| 偵測範圍 | 中等 |
| 移動速度 | 慢 |
| 危險程度 | ⭐⭐ |

**特點**：
- 會追蹤玩家位置
- 進入攻擊範圍會造成傷害
- 可被擊退

### 遠程殭屍 (Ranged Zombie)
| 屬性 | 說明 |
|------|------|
| 攻擊方式 | 遠程射擊 + 近戰 |
| 偵測範圍 | 較大 |
| 移動速度 | 中等 |
| 危險程度 | ⭐⭐⭐ |

**特點**：
- 會發射投射物攻擊玩家
- 保持距離時會持續射擊
- 靠近後切換為近戰

### Boss（遠程殭屍 Boss 模式）
| 屬性 | 說明 |
|------|------|
| 生命值 | 極高 |
| 攻擊方式 | 遠程 + 近戰 |
| 危險程度 | ⭐⭐⭐⭐⭐ |

**特點**：
- 進入範圍時顯示專屬血條
- 需要大量彈藥擊殺
- 建議先清理周圍小怪

## 🚪 場景互動

### 門 (Door)
| 狀態 | 操作 |
|------|------|
| 未上鎖 | 靠近自動開啟 |
| 上鎖 | 需要對應鑰匙 + 完成 QTE |

### QTE（Quick Time Event）
當靠近上鎖的門且持有對應鑰匙時：
1. 畫面出現圓形 QTE 指示器
2. 指針會旋轉一圈
3. 在指針經過綠色區域時按下**左鍵**
4. 成功 = 門解鎖並開啟
5. 失敗 = 需要退出區域再重新嘗試

**注意**：QTE 進行中被敵人攻擊會直接失敗！

### 寶箱 (Chest)
- 可能上鎖，需要鑰匙
- 內含彈藥、血包等補給

### 傳送點 (Teleport)
- 將玩家傳送到其他位置
- 用於場景切換

## 📦 道具系統

### 鑰匙 (Key)
- 不同顏色/ID 對應不同的門
- 撿取後自動裝備
- 使用後消耗（可設定）

### 補給品
| 道具 | 效果 | 撿取條件 |
|------|------|----------|
| 血包 | 回復生命值 | 血量 ≤ 70% |
| 護甲 | 回復護甲值 | - |
| 小電池 | 回復電量 20 | - |
| 中電池 | 回復電量 40 | - |
| 大電池 | 回復電量 60 | - |
| 煤油 | 回復煤油 | - |

### 彈藥
| 道具 | 對應武器 |
|------|----------|
| 手槍彈藥 | 手槍 |
| 步槍彈藥 | 步槍 |
| 箭矢 | 弓箭 |

---

## 🔦 光源系統

### 手電筒 (Flashlight)
- 消耗電池
- 照亮前方區域
- 電池耗盡會熄滅

### 提燈 (Lantern)
- 消耗煤油
- 360度照明
- 範圍較小但穩定

### 房間燈光
- 部分房間有開關
- 可照亮整個房間

## 🗺️ 遊戲流程

### 關卡結構
```
開始 → 探索場景 → 尋找鑰匙 → 解鎖門 → 
       ↓
  擊殺敵人 ← 避開陷阱
       ↓
收集補給 → 前進到下一區域 → 到達終點
       ↓
   勝利畫面
```

### 存活策略

1. **資源管理**
   - 不要浪費彈藥在不必要的敵人上
   - 優先確保手電筒有電
   - 血量低於 50% 時優先尋找血包

2. **戰鬥技巧**
   - 保持移動，避免被包圍
   - 擊退敵人後趁機拉開距離
   - Boss 戰前確保彈藥充足

3. **探索建議**
   - 仔細搜索每個房間
   - 注意隱藏的鑰匙和道具
   - 記住補給點位置

## 🎮 遊戲畫面

### 主要 UI 元素
| 位置 | 顯示內容 |
|------|----------|
| 左上 | 生命值條（紅→綠） |
| 左上 | 護甲值條 |
| 右上 | 彈藥數量 |
| 右下 | 電池/煤油剩餘 |
| 中間 | 背包快捷欄 |

### 特殊 UI
- **Boss 血條**：Boss 戰時出現在畫面上方
- **QTE 指示器**：解鎖門時出現在畫面中央
- **指南針/雷達**：顯示目標方向

---

## ⚠️ 遊戲提示

1. **手電筒很重要** - 在黑暗中你什麼都看不到
2. **聽聲辨位** - 殭屍會發出呻吟聲，注意周圍
3. **不要戀戰** - 有時逃跑比戰鬥更聰明
4. **QTE 要專心** - 進行 QTE 時確保周圍安全
5. **探索每個角落** - 隱藏的補給可能救你一命

---

## 🏆 勝利條件

- 到達關卡的 **Exit（出口）** 區域
- 進入後自動進入勝利畫面

## 💀 失敗條件

- 生命值歸零
- 進入死亡畫面，可選擇重新開始
